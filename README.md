# Sveta website draft

Это стартовый лендинг для уроков английского.

## Что уже есть
- первый экран
- блок "о себе"
- форматы занятий
- цены
- ссылки на каналы/сообщества
- контакты

## Что нужно заменить
В `index.html` замени:
- `https://calendly.com/your-link` — ссылка на запись
- цены `€XX`
- ссылки на Telegram / Instagram / YouTube
- `https://t.me/yourusername`
- `hello@example.com`
- текст под свои реальные форматы

## Как открыть
Просто открой `site/index.html` в браузере.
